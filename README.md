# Trident-Revamp-Repository
This public repository only contains the documentation and license files for the "Trident Revamp" resource pack. It does NOT contain private model or texture sources.

A vanilla-like 3D redesign for Minecraft tridents.

## Features
- Fresh vanilla-style 3D trident model
- Custom textures
- Fully compatible with vanilla Minecraft
